## Observations

### CPU
The CPU is the brain of the computer. It processes instructions and controls system operations.

### RAM
RAM is temporary memory used for running programs. More RAM allows better multitasking.

### Storage Device
My computer uses an SSD. It is faster than HDD and improves performance.

### Input Devices
- Keyboard
- Mouse
- Microphone

### Output Devices
- Monitor
- Speakers
- Printer

## What I Learned
I learned that all hardware components work together to make a computer function efficiently. The CPU processes data, RAM stores temporary information, and storage saves permanent data.
